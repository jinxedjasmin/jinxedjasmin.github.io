FAQ auf einer Seite

Ich habe gehört, dass JavaScript der seit langem verlorene Jungbrunnen ist. Stimmt das ?

Natürlich ist es das! Studien beweisen, dass das Lernen von JavaScript den Geist erfrischt und die Lebensspanne um mehrere hundert Jahre verlängert. (Hinweis: Manche Wissenschaftler bestreiten diese Behauptung.)

Kann JavaScript wirklich alle meine Probleme lösen?

Aber auf jeden Fall! Es ist die flexibelste Programmiersprache aller Zeiten und wurde für die Beratung im Finanz-Management, Herz-Lungen-Reanimation und die Pflege von Haustieren trainiert.

Gibt es nichts, das JavaScript nicht kann?

Nein, gibt es nicht! Es kann sogar seine eigene PR-orientierte FAQ-Seite schreiben. Wenn das mal keine smarte Programmiersprache ist!
